How I Installed Vue.js

I used Vue.js by including its CDN link directly in the HTML file. This method is simple, especially when deploying on GitHub Pages. I added the following line to my `` section: 

<script src="https://cdn.jsdelivr.net/npm/vue@3.2.47/dist/vue.global.min.js"></script>

This allowed me to use Vue.js features directly without setting up a build tool like Webpack or Vite.

How I Built This Page

I structured this page with three main sections:

  • Header: Displays the page title and provides an introduction.
  • Vue.js Features: A section powered by Vue.js to dynamically display a message and interact with users.
  • Documentation: Details the installation process and challenges faced.

Challenges Faced

One of the main challenges was ensuring compatibility with GitHub Pages, as Vue.js often requires a build tool for complex projects. By using the CDN version of Vue.js, I was able to avoid these issues.

Another challenge was ensuring that the page was fully responsive. I solved this by testing the layout on different screen sizes and adjusting the CSS to use percentages and `max-width`.
